HTML Emails and Newsletters

Entering the web development world in recent years, I escaped the pain of building websites using tables for layout. But the ability to code emails was essential for me to learn.

So I went back in time and taught myself how to build sturdy emails to code like it’s 1999 - using tables, HTML attributes, and inline styles. I hope you enjoy my email designs.

After learning to code emails using techniques familiar to coders ten years ago, I was delighted when introduced to Ink is a responsive HTML email framework by Zurb. Ink tries to make it easy to build custom and responsive HTML emails that work on all devices.

Overnight, it was now a pleasure to code emails. I hope you enjoy my designs and would welcome your feedback.

Animation

I've recently enjoyed working with GreenSock, the light, but powerful JavaScript framework which makes it easy to animate HTML elements. GreenSock is going to help me write clean, efficient animation. The ease of use, clear documentation and commmunity support has made GreenSock a benchmark product.

I liked GreenSock to such an extent that I've created a website showcasing my work. I look forward to adding to my collection of work in the coming weeks.
